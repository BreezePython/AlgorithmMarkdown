# 14. 最长公共前缀
> https://leetcode-cn.com/problems/longest-common-prefix/
> 
> 难度：简单

## 题目：

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

## 示例：

示例 1：

输入：strs = ["flower","flow","flight"]

输出："fl"

示例 2：

输入：strs = ["dog","racecar","car"]

输出：""

解释：输入不存在公共前缀。

## 分析

只要你熟悉zip方法的使用，这种题简直是塞牙缝的...

## 解题：

```python
class Solution:
    def longestCommonPrefix(self, strs):
        ret = ''
        for i in zip(*strs):
            if len(set(i)) == 1:
                ret += i[0]
            else:
                break
        return ret
```
